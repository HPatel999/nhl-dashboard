# NHL Analytics Dashboard

A web application that lets you explore NHL teams and players with up-to-date statistics and information. You can view team rosters, player profiles, game stats, and other hockey data in one place. This project builds on an earlier analytics dashboard I created in [Power BI](https://github.com/HPatel999/NHL-PowerBi-DashBoard), expanding the concept into a full web experience

**Live Application:**
[https://nhl-dashboard-psi.vercel.app](https://nhl-dashboard-psi.vercel.app/) 

---

## Key Features

### Team Pages

* Select any NHL team to explore its full profile
* View current roster and player positions
* Access team schedule and upcoming games
* Comprehensive team-wide statistics including:

  * Road vs Home performance
  * Goals For vs Goals Against
  * Faceoff breakdowns
  * Shooting type analysis
  * Special teams performance
* Interactive visualizations and charts for quick insights
* Automatic team color theming for a cohesive UI experience

### Player Pages

#### Skaters

* Career and seasonal production stats
* Faceoff breakdowns (for centers)
* Shooting analysis and scoring tendencies
* Deployment statistics (e.g., shifts, ice time)
* Advanced metrics including Corsi and Fenwick
* Special teams contributions (power play, penalty kill)
* Interactive tables and charts for performance trends

#### Goalies

* Career and seasonal performance tables
* Workload vs efficiency graphs ( total shots against vs save percentage)
* Save percentage breakdowns by:

  * Game state (even strength, power play, shorthanded)
  * Days of rest
  * Starting vs relieving

This dashboard consolidates a wide range of NHL data into an interactive, easy-to-navigate interface for both teams and individual players.
